1)	Создайте собственную функцию для вычисление факториала числа. Создайте 
словарь CACHE_FACTORIAL. В качестве ключа используйте число а в качестве 
значения его факториал. Покажите работу кэша добавив нагрузку в виде задержки 
в 3 секунды при вычислении факториала, если числа нет в вашем словаре. 

from time import sleep

CACHE_FACTORIAL = {}


def factorial(some_int):
    if some_int not in CACHE_FACTORIAL:
        sleep(3) 
        result = 1
        for i in range(1, some_int + 1):
            result *= i
        CACHE_FACTORIAL[some_int] = result
    return CACHE_FACTORIAL[some_int]


print(factorial(5))  
print(factorial(5))  
print(factorial(10))

2)	С помощью декоратора @lru_cache закешируйте функцию которая рекурсивно 
считает сумму положительных чисел списка. 

@lru_cache()
def sum_positive_numbers(numbers_tuple):
    if len(numbers_tuple) == 0:
        return 0
    elif numbers_tuple[0] > 0:
        return numbers_tuple[0] + sum_positive_numbers(numbers_tuple[1:])
    else:
        return sum_positive_numbers(numbers_tuple[1:])


numbers_list = [1, 2, 3, 4, -5]
numbers_tuple = tuple(numbers_list)
result = sum_positive_numbers(numbers_tuple)
print(result)
